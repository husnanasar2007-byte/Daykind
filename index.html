Daykind

Daykind is an AI-based daily reflection and well-being app built using MIT App Inventor.

Problem

Many people struggle to reflect on their emotions and daily experiences in a healthy way.

Solution

Daykind allows users to enter short daily reflections and receive meaningful AI-generated insights.

Key Features

  • Simple daily reflection input
  • AI-generated emotional insights
  • Calm, minimal user interface

Daily Reflection

AI Usage

Google Gemini API is used to generate insights based on user input.

Tools & Technologies

  • MIT App Inventor
  • Google Gemini API
  • Canva (UI & presentation)

API Key Disclaimer

This project uses the Google Gemini API. The API key is NOT included for security reasons. Users must add their own API key while testing or deploying the app.
